PLACEHOLDER IMAGES GUIDE
========================

This folder should contain the following images for your real estate project website.
Replace these placeholder references with your actual project images.

   - activity-2.jpg (800x600px) - Activity image 2
   - activity-3.jpg (800x600px) - Activity image 3

IMAGE GUIDELINES:
-----------------

File Formats:
- Preferred: JPG for photos, PNG for images with transparency
- Consider WebP format for better compression (modern browsers)

Optimization:
- Compress images before uploading (use TinyPNG, ImageOptim, or similar)
- Target file sizes: Hero (500KB max), Other images (300KB max)

Dimensions:
- Maintain aspect ratios for best display
- Use higher resolution images for better quality on retina displays
- The website is responsive and will scale images appropriately

Naming Convention:
- Use lowercase letters
- Use hyphens (-) instead of spaces
- Be descriptive: exterior-view.jpg, interior-lobby.jpg, etc.

QUICK START:
------------

To get your website running quickly:

1. Download 13 images total (sizes mentioned above)
2. Rename them according to the list above
3. Place them in this images/ folder
4. Open index.html in your browser
5. Your website will display with your images!
